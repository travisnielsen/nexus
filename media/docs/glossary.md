# Glossary

Quick reference for abbreviations and short names used in this project.

## Terms

| Term       | Full Name                             | Description                                              |
| ---------- | ------------------------------------- | -------------------------------------------------------- |
| A2A        | Agent-to-Agent                        | Protocol for inter-agent communication                   |
| ADR        | Architecture Decision Record          | Documented architectural decisions                       |
| AG-UI      | Agent–User Interface                  | SSE-based protocol connecting agents to frontend UIs     |
| API        | Application Programming Interface     | HTTP endpoints, SDK interfaces                           |
| CI/CD      | Continuous Integration/Delivery       | Automated build and deploy pipelines                     |
| CLI        | Command-Line Interface                | Terminal-based tools (uv, npm, az)                       |
| DuckDB     | DuckDB                                | In-process SQL engine used by the MCP server             |
| IaC        | Infrastructure as Code                | Terraform templates                                      |
| MAF        | Microsoft Agent Framework             | Agent orchestration framework                            |
| MCP        | Model Context Protocol                | Protocol for exposing data and tools to AI agents        |
| MSAL       | Microsoft Authentication Library      | Azure AD token acquisition library                       |
| PR         | Pull Request                          | Code review request on GitHub                            |
| RBAC       | Role-Based Access Control             | Azure permission model                                   |
| SSE        | Server-Sent Events                    | Streaming protocol for real-time UI updates              |
| TLS        | Transport Layer Security              | Encryption protocol (require 1.2+)                       |

## File Types

| Extension    | Description                            |
| ------------ | -------------------------------------- |
| `.agent.md`  | GitHub Copilot custom agent definition |
| `.prompt.md` | GitHub Copilot reusable prompt file    |
