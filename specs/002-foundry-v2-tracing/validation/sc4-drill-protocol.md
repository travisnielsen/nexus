# SC-004 Incident Drill Protocol

## Goal

Identify failing step (turn/tool/A2A) within 5 minutes in at least 90% of sampled failures.

## Drill Procedure

1. Generate synthetic failures (tool failure, A2A failure, A2A timeout).
2. Start timer when incident ticket opens.
3. Use Foundry portal trace and dashboard to identify failure step.
4. Record detection time and classification accuracy.
